Amara's Boolean Expression Solver

To use this, input an equation in the form of AB * CD = EFGH into the text box.
The truth table will be generated, and the simplified/reduced expressions will be displayed below.
AB * CD = EFGH is a valid expression,
Which represents a 2x2 multiplier, the number of variables for each term, represents the number of bits in each term.
The number of variables on the left side, are the number of input bits.
The number of variables on the right side, are the number of output bits.
